# Digital Footprint Analyzer

## Overview

Digital Footprint Analyzer is a responsive web application built using HTML, CSS, and JavaScript. It analyzes basic user information such as name, email address, and username to identify potential privacy risks and provides recommendations for creating a safer online identity.

---

## Features

- Analyze username, email, and full name
- Generate a digital footprint risk score
- Identify potential privacy risks such as:
  - Username contains personal information
  - Username contains numbers
  - Username is too short
  - Email contains the user's name
  - Username and email are identical
- Display risk level (Low, Medium, High)
- Generate personalized privacy recommendations
- Responsive design for desktop, tablet, and mobile devices
